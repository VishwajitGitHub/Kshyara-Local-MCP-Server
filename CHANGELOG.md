# Changelog 📝

All notable changes to this project will be documented in this file.

## [5.0.0] - 2026-03-26
### Added
- **32,000+ Tools:** Massive expansion of the tool library across 10 domains.
- **150+ Workflows:** Automated enterprise pipelines for complex task orchestration.
- **Strict License System:** New tiered licensing (Basic, Pro, Premium, Enterprise).
- **Remote Validation:** License keys are now validated against a remote server.
- **Enterprise Subsystems:** Added Logger, CacheManager, SecurityAuditor, and MetricsCollector.

### Changed
- **Refactored Engine:** Optimized server code to handle 32k+ tools efficiently in a single file.
- **License Update:** Switched from MIT to a Custom Commercial License to protect intellectual property.
- **Documentation:** Updated README and added SETUP.md for better onboarding.

## [4.0.0] - 2026-03-25
### Added
- **5,000+ Tools:** Initial large-scale tool generation.
- **SSE Support:** Added Server-Sent Events transport for web integrations.

### Changed
- **MCP SDK:** Upgraded to the latest Model Context Protocol SDK.

## [3.0.0] - 2026-03-24
### Added
- **Initial Release:** Core MCP server functionality with basic tool registration.
- **Stdio Transport:** Standard input/output communication for AI clients.
